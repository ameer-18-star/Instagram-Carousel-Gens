# Carousel Generators
JSON Data to Instagram Caarousel Post Generators.
All the files in this repository have different design and you have to copy JSON data in these files and use AI to get JSON data related to your post and then paste JSON in this HTML file and click on Generate Carousel Post Button to generate Carousel Post. Then you can download the whole post in .zip file or you can download images 1-by-1 in .png or .jpg files. Your carousel is ready to post.
